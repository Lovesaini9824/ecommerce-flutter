## Tech Stack

### Frontend (Flutter)
- **Framework**: Flutter 3.2+
- **State Management**: Provider
- **HTTP**: `http` + `shared_preferences`
- **UI**: Glassmorphic cards, gradients, animations
- **Assets**: Google logo, shimmer loading

### Backend (Node.js)
- **Runtime**: Node.js v20
- **Framework**: Express.js
- **Database**: MongoDB
- **Auth**: JWT
- **CORS**: Enabled for Web, Emulator, Real Device
